# Ocula Blink Analyst | 智能眼部眨眼分析平台

![Version](https://img.shields.io/badge/version-1.0.0-blue.svg) ![AI](https://img.shields.io/badge/Powered_by-Deep_Learning-indigo)

欢迎使用 **Ocula Blink Analyst**。这是一个专业的眼部健康辅助分析工具，专为眼科研究人员和医疗专业人士设计。系统利用先进的计算机视觉与深度学习技术（U-Net），从面部视频中自动提取眼睑运动数据，精准识别并量化眨眼模式。

---

## 🌟 核心功能

### 1. 智能视频分析
只需上传一段包含面部的视频，系统即可自动进行逐帧分析。无需手动标记，算法会自动分割眼部区域并计算眼睑闭合程度。

### 2. 精准眨眼分类
系统能够区分并统计不同类型的眨眼，这对于评估干眼症（DED）和其他眼表疾病至关重要：
*   **完全眨眼 (Complete Blink)**: 眼睑完全闭合，有效清洁眼球表面。
*   **不完全眨眼 (Incomplete Blink)**: 眼睑未完全接触，可能导致泪膜破裂。

### 3. 可视化数据仪表盘
*   **实时波形图**: 直观展示眼睑面积随时间的变化曲线。
*   **关键指标统计**: 自动计算不完全眨眼比例、眨眼频率等核心指标。
*   **详细报告**: 提供归一化的数据分析结果。

在使用前，您需要阅读并签署《数据使用知情同意书》。我们严格遵循伦理规范，确保所有上传的视频数据仅用于当次分析，保障受试者隐私。

### 第三步：上传视频
*   点击上传区域或拖拽视频文件。
*   支持格式：`.mp4`, `.mov`, `.avi`。
*   **建议**: 视频光线充足，受试者头部保持相对静止，眼部清晰可见。

### 第四步：查看报告
分析完成后，系统将生成交互式图表。您可以：
*   查看每一次眨眼的具体数值。
*   对比完全眨眼与不完全眨眼的分布。
*   查看基于归一化面积的统计摘要。

## 🔬 技术原理

本平台后端采用经过大量真实样本训练的 **U-Net 分割模型**。
1.  **预处理**: 视频抽帧与人脸关键点定位。
2.  **推理**: 使用 PyTorch 加载 `best_model_Unet.pth` 权重，对眼睑区域进行像素级分割。
3.  **后处理**: 使用 pandas 与 numpy 进行信号处理，设定阈值（如 Drop Threshold）来判定眨眼事件。
